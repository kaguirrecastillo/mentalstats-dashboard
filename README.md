MentalStats Dashboard

End-to-end analytics mini-project: data cleaning in Python + interactive dashboard (Looker Studio).

What this project does
- Loads public health indicators (suicide rates + mental health resources)
- Cleans and reshapes data into analysis-ready tables
- Exports `reports/cleaned.csv`
- Visualizes trends with filters by year / country / demographic group (Looker Studio)

Repo structure
- `notebooks/` — analysis + cleaning notebooks
- `src/` — reusable scripts (optional pipeline)
- `reports/` — cleaned outputs used by dashboard
- `images/` — screenshots for README
